## 📋 О проекте

Это **полный, продуманный, готовый план разработки** кроссплатформенного мобильного приложения для сети компьютерных клубов **BBplay**. 

Приложение позволяет посетителям:
- ✅ Авторизоваться и управлять аккаунтом
- ✅ Пополнять баланс
- ✅ Бронировать компьютеры в клубах
- ✅ Общаться с чат-ботом (AI на базе Qwen)
- ✅ Смотреть новости из VK
- ✅ Находить идеальные места с умным подбором по времени

**Все спроектировано для максимально эффективной разработки через AI (Cursor + Claude 3.5 Sonnet).**

## 🚀 Быстрый старт (5 минут)

### 1️⃣ Установи Cursor IDE
```bash
# Скачай с https://www.cursor.sh/
# Установи на свою платформу (Windows, Mac, Linux)
```

### 2️⃣ Создай Flutter проект
```bash
flutter create bbplay --org com.blackbears
cd bbplay
```

### 3️⃣ Открой проект в Cursor
```bash
cursor .
```

### 4️⃣ Используй готовые промпты
Скопируй текст из `prompts.md` (Промпт 1) в Cursor Compose и жди генерирования кода

### 5️⃣ Запусти на Web
```bash
flutter run -d chrome
```

**Готово! Дальше следуй `QUICK_START.md`**

## 🧪 Тестирование

### Способы тестирования

| Способ | Скорость | Удобство | Когда использовать |
|--------|----------|----------|-------------------|
| **Web** ⚡⚡⚡ | Очень быстро | Отлично | Разработка UI/логики |
| **Android Emulator** ⚡⚡ | Быстро | Хорошо | Полное тестирование |
| **iOS Simulator** ⚡ | Медленно | Хорошо | Финальная проверка |

### Рекомендуемый порядок

1. **Разработка** → Web (flutter run -d chrome)
2. **Интеграция** → Android Emulator (flutter run)
3. **Финал** → iOS Simulator (flutter run -d ios)

### Запуск на эмуляторе

```bash
# Android
flutter emulators --launch Pixel_6_API_31
flutter run

# iOS (Mac только)
open -a Simulator
flutter run -d ios

# Web
flutter run -d chrome
```
