## 📋 О проекте


Приложение позволяет посетителям:
- ✅ Авторизоваться и управлять аккаунтом
- ✅ Пополнять баланс
- ✅ Бронировать компьютеры в клубах
- ✅ Общаться с чат-ботом (AI на базе Qwen)
- ✅ Смотреть новости из VK
- ✅ Находить идеальные места с умным подбором по времени

**Все спроектировано для максимально эффективной разработки через AI (Cursor + Claude 3.5 Sonnet).**

---

## 🎯 Особенности

### 🚀 Разработка через AI
- 15 готовых AI-промптов для генерации 100% кода
- Экономия времени: **65-75%** по сравнению с ручной разработкой
- Ожидаемое время разработки: **15-16 часов** (против 40-50 часов вручную)
- Все промпты оптимизированы и протестированы

### 📱 Кроссплатформенность
- **Android** (APK для эмулятора и реальных устройств)
- **iOS** (IPA через TestFlight или App Store)
- **Web** (Flutter Web на GitHub Pages)
- Responsive дизайн для всех размеров экранов

### 🎨 Дизайн и UX
- Тёмная тема в стиле **BlackBears** (чёрный + неон синий/фиолетовый)
- Material 3 компоненты
- Гладкие анимации и transitions
- Интуитивная навигация с Bottom Tab Bar

### 🔐 Безопасность и масштабируемость
- Firebase Authentication (Email + Google)
- Firestore для хранения данных с правильными Rules
- Cloud Messaging для push-уведомлений
- Обработка ошибок и offline режима

### 🤖 AI-интеграция
- **Qwen Chat API** для умного чат-бота с базой знаний про клубы
- **VK API** для загрузки новостей из сообщества

### 📚 Полная документация
- `QUICK_START.md` — как начать за 5 минут
- `PLAN.md` — полный план разработки со всеми фазами
- `architecture.md` — детальная архитектура приложения
- `prompts.md` — 15 готовых AI-промптов
- `TESTING_STRATEGY.md` — как тестировать на всех платформах
- `CHECKLIST.md` — финальный контроль качества

## ✨ Основной функционал

### 🔐 Авторизация
- Email + Password вход/регистрация
- Google Sign-In
- Восстановление пароля
- Session management

### 👤 Профиль
- Просмотр баланса
- История платежей
- История бронирований
- Редактирование профиля

### 💰 Пополнение баланса
- Выбор способа оплаты (карта, Яндекс, Сбербанк)
- Mock-платежи для тестирования
- История операций

### 📅 Бронирование ПК
- Выбор клуба по городу
- Выбор зала в клубе
- Календарь с доступностью
- Выбор времени (30 мин - 4 часа)
- Выбор компьютера
- Расчёт стоимости
- Подтверждение и создание бронирования

### 💬 Чат-бот (AI)
- Вопросы-ответы на базе Qwen API
- База знаний про клубы, тарифы, правила
- История чата
- Красивое отображение сообщений

### 📰 Новости
- Загрузка постов из VK (BBPlay группа)
- Фильтры (посты, события, акции)
- Auto-refresh каждые 30 минут

### 🌟 Бонус функции
- ⭐ Умный подбор свободных мест по времени
- ⭐ Календарь с отметкой доступности
- ⭐ Push-уведомления о подтверждении
- ⭐ Тёмная тема BlackBears
- ⭐ Локализация на русский
